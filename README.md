# rpi-rfid
RPi interface with MFRC522 via big-bang SPI

Currently only supports reading UID.

## Pin

Pin # on RPi | Pin on MFRC522 | GPIO #
--- | --- | ---
35 | SDA | GPIO19
36 | MISO | GPIO16
37 | RST | GPIO26
38 | MOSI | GPIO20
39 | GND | GND
40 | SCLK | GPIO21
17 | 3.3V | 3.3V
